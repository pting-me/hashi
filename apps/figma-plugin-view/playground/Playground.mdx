# Component Playground

This section is intended to let you try out components components exported by the plugin.

## Instructions

To test your components, drop it in `apps/figma-plugin-view/playground/components`.

Then, update the imports for the story in `apps/figma-plugin-view/playground/Playground.stories.tsx`.

Before

```tsx
import Component, { type Props } from './exampleComponents/Button';
```

After

```tsx
import Component, { type Props } from './components/MyComponent';
```
